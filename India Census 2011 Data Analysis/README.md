## Overview
This project is based on India Census 2011 data. It is used for data analysis to understand population structure, literacy, employment, and religious distribution across India.

## Dataset Features
- Population (Total, Male, Female)
- Literacy data
- Workers (Male & Female)
- Cultivator & Agricultural workers
- Religion-wise population (Hindus, Muslims, Christians, Sikhs, Buddhists, Jains)
- Education levels (Secondary, Higher, Graduate)
- Age groups (0–29, 30–49, 50+)
- State and District names


## Skills Demonstrated
- Data Cleaning & Preprocessing  
- Pandas DataFrame operations  
- GroupBy & Aggregation  
- Filtering & conditional selection  
- Handling large structured datasets  
- Statistical summarization  
- Insight generation from data  
- Column manipulation (Prefix & Suffix using add_prefix() and add_suffix())  


## Key Analysis
- State-wise total population
- Religion-wise population analysis
- Male vs Female workers comparison
- Filtering districts (New Delhi, Lucknow, Jaipur)
- GroupBy operations for state-level insights
- Data indexing and formatting

## Key Insights
- Uttar Pradesh has the highest population
- Maharashtra has highest male workers
- Punjab has highest Sikh population
- Strong variation in literacy across states
